## Building with CMake
```
$ sudo apt-get install cmake libsdl2-dev
$ git clone https://github.com/abatef/c8emu.git
$ cd c8emu
$ cmake CMakeLists.txt
$ make
```

## Usage

```
c8emu -rom ./path/to/rom/file
```
